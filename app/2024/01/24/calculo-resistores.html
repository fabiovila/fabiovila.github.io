Valor Tolerância

Valores comuns de resistores

  • 330Ω é um valor comum para alimentar um LED em 5V com corrente de 10mA
  • 1kΩ é um valor comum para alimentar um LED em 5V com corrente de aproximadamente 3mA
    • Esse valor é usado pelo Arduino em série com os LEDs
  • 1k2Ω é um valor comum para alimentar um LED em 12V com corrente de aproximadamente 3mA
  • 10KΩ é um valor comum para Pull Up e Pull Down
    • Esse valor é usado pelo Arduino como Pull Up
  • 510Ω é usado pela placa BluePill (STM32 a 3,3 Volts) como resistor de LEDs
  • 0.18Ω e 0.01Ω são valores comuns para resistores Shunt ( sensores de corrente )
  • 1MΩ é normalmente usado em paralelo com cristais de clock

Valores Comerciais

O comércio não comercializa todos os valores possíveis. A indústria padronizou alguns valores que são multiplicados pela 3° Faixa, por exemplo, o valor 8.2 pode ser 8.2Ω, 82Ω, 820Ω, 8,2KΩ e assim por diante. Na tabela abaixo pode-se conferir esses valores.

Padrão E6, E12 e E24    
1.0 1.1 1.2 1.3
1.5 1.6 1.8 2.0
2.2 2.4 2.7 3.0
3.3 3.6 3.9 4.3
6.8 7.5 8.2 9.1

Esses valores são para resistores de tolerância entre 5% e 20%. Para tolerâncias menores entre 1% e 0.1% a quantidade de valores comerciais é maior e neste caso uma faixa adicional é usada para indicar a maior precisão de valores.
